# nostr-ni57721-bot
A Nostr bot for making the world more peaceful.  
When the bot detects key words in other's posts, it replies something to the
posts.

## Usage
Specify the private key and relays you use in .env.

Here's an example:
```.env
export NOSTR_NSEC="nsec..."
export NOSTR_RELAYS="
  wss://example.com/foo/
  wss://example.com/bar/
  wss://example.com/baz/
"
```
